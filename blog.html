Discuss the scope of var, let, and const

Ans: Var can be accessed through the whole script of a js file. That's why its' scope is called global scope. It can be used before initialization. It can be declared more than once within the same script. variable declared with var will be hoisted .
let cannot be used before the initialization. Its' scope is called block scope. It can be used only with in block scope or in function scope. Outside of the block it cannot be accessed. It cannot be declared with the same value more than once. But the value of the variable declared with the let can be changed. variable declared with let are also hoisted but not initialized. let allows reassignment of value. const also cannot be used before initialization. It also posses block scope. out side of the block or function scope it cannot be accessed. It didn't allow reassignment of value.The value within const cannot be changed.

Tell us the use cases of null and undefined

Ans: Null is a value that doesn't possess anything else. It is given deliberately when there is no value to give. On the other hand undefined is used when any data or value is not found or don't know what is the actual value is. undefined is used on unintentional behavior of data.
there are 8 ways to get Undefined :
1.Variable that is not initialized will give undefined.
2.Function with no return.
3.Parameter that is not passed will be undefined.
4.If return has nothing on the right side will return undefined.
5.Property that doesn't exists on object will give undefined.
6.Accessing array elements outside of the index range will give undefined.
7.Deleting an element inside an array will create undefined .
8.Set a value directly to undefined.

What do you mean by REST API?

Ans: The full form of REST API is Representational State Transfer Application Programming Interface. Those are some set of rules that ensure how the data should passes through internet from web page to server. Those are not any kind of protocol, they are simply designed for proper functioning of webs services.
